# DNS Records
This script takes the website name as input and returns its dns records.

### Tech Stack:
+ Python

### Libraries used:
+ dnspython

###  Pre-requirements:
+ `pip install -r requirements.txt`

### To execute the project:
+ Run `dns_record.py`
