# Movie Information Scrapper
This script obtains movie details by scraping IMDB website.

### Tech Stack:
+ Python

### Libraries used:
+ beautifulsoup4
+ requests

###  Pre-requirements:
+ install `pip install -r requirements.txt`

### To execute the project:
+ Run `python movieInfoScraper.py`
